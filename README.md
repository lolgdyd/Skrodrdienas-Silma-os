# Skroderdienas-Silmačos
Skroderdienas Silmačos ir viena no slavenākajām un visvairāk iestudētajām lugām latviešu literatūrā un teātrī. Tās autors ir Rūdolfs Blaumanis. Luga sarakstīta 1901. gadā, bet pirmizrāde notika 1902. gada 30. janvārī Rīgas Latviešu teātrī.


Rūdolfs Blaumanis bija ievērojams latviešu rakstnieks, dramaturgs un žurnālists, viens no nozīmīgākajiem latviešu literatūras klasiķiem.

Dzimšana

Rūdolfs Blaumanis dzimis 1863. gada 1. janvārī Ērgļu pagasta “Brakos”, Vidzemē (tagad — Braku muzejs).

Nāve

Viņš nomira 1908. gada 4. septembrī Somijā, Takaharju sanatorijā no tuberkulozes. Rakstnieks apglabāts Rīgā, Meža kapos.

Galvenie darbi

Svarīgākie Rūdolfa Blaumaņa darbi:

stāsts Raudupiete

komēdija Trīnes grēki

2 interesanti fakti

Rūdolfs Blaumanis rakstīja ne tikai latviešu valodā, bet arī vācu valodā, jo tajā laikā tā bija plaši lietota Baltijā.

Luga Skroderdienas Silmačos ir viena no visbiežāk iestudētajām lugām Latvijā, īpaši Jāņu laikā.
